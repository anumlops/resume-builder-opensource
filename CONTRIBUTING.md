# Contributing to Resume Builder

We welcome contributions! Whether it's bug fixes, features, or documentation, your help is appreciated.

## Getting Started

1. Fork the repo
2. Clone your fork
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Run tests: `npm test`
6. Commit: `git commit -am 'Add feature'`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request

## Areas We Need Help With

- **Verb Libraries**: Industry-specific power verbs (Finance, Tech, Healthcare, etc.)
- **Templates**: Resume variations for different career stages
- **Testing**: Expand test coverage for bullet generation
- **Documentation**: Improve docs and add tutorials
- **Translations**: Translate for non-English speakers
- **UI**: Web interface for non-CLI users

## Code Style

- Use TypeScript for all new code
- Follow existing patterns
- Add unit tests for new features
- Document complex functions
